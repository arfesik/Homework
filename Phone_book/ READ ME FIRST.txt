Курсовая работа по СИ: Программа "Телефонный справочник"

1. Файл phonebook.txt содержит готовый заполненный вариант телефонной книги, в программу данную книгу можно загрузить выбрав пункт "4. Read from FILE". 

----- ФАЙЛ phonebook.txt НЕОБХОДИМО ПОМЕСТИТЬ В ДИРЕКТОРИЮ ГДЕ НАХОДИТСЯ ИСПОЛНЯЕМЫЙ ФАЙЛ! -----

2. Пункт меню "1. ADD person..." добавляет запись в телефонную книгу если она была подгружена из файла и создает телефонную книгу в памяти компьютера с нуля, если из файла данные не загружались.

Формат записи 1. Имя: 
              2. Фамилия:
              3. Номер: x(xxx)xxx-xx-xx или x-xxx-xxx-xx-xx или xxxxxxxx 

3. Пункт меню "2. DISPLAY..." отображает на экран текущее состояние телефонной книги.

4. Пункт меню "3. Write to FILE" записывает в файл все записи в телефонной книге.

5. Пункт меню "4. Read from FILE" считывает в память программы все записи из файла. (Для того чтобы отобразить их нажмите "2. DISPLAY")

6. Пункт меню "5. Search person" осуществляет поиск по выбранному параметру: имя, фамилия, номер телефона и отображает найденные контакты. Вы можете искать как по полному имени/фамилии/номеру, так и по части имени/фамилии/номера (поиск по вхождению)

7. Нажмите пункт "6. Exit" чтобы завершить работу
